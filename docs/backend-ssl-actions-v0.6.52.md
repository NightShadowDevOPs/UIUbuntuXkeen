# v0.6.52 — backend SSL actions polish

## Что добавлено
- На экране `Хосты 3x-ui` разделены два явных действия: **Проверить сейчас** и **Обновить SSL-кэш**.
- Карточка SSL-проверок провайдеров теперь тоже показывает оба действия, чтобы не приходилось прыгать между экранами.
- В статусных бейджах экрана `Хосты 3x-ui` теперь выводятся ближайшая проверка и состояние последнего job.

## Что исправлено
- `backend/scripts/install.sh` теперь не только копирует файлы, но и **явно перезапускает** `ultra-ui-ubuntu-backend.service`, если сервис уже был запущен.
- Убрана ловушка, когда код backend уже обновился до новой версии, а процесс `uvicorn` продолжал отвечать старой версией до ручного `systemctl restart`.

## Что проверить на сервере
1. После `git pull` и `./backend/scripts/install.sh` `curl http://127.0.0.1:18090/api/version` должен вернуть `0.6.52` без ручного рестарта.
2. В `Setup` активным остаётся backend `ubuntu-service`.
3. На странице `Хосты 3x-ui` обе кнопки должны быть доступны:
   - `Проверить сейчас`
   - `Обновить SSL-кэш`
4. После нажатия статусные бейджи должны обновляться без возврата к fallback.
