# База шагов и хода работ

Актуально для релиза: **v0.6.43**

## Выполненные шаги
1. Взята за базу очищенная версия `v0.6.42`.
2. Проверены реальные маршруты и разделы: `Users` и `Xui Hosts` уже существуют в приложении и висят в меню.
3. Проверен текущий runtime contour проекта: UI обслуживается `mihomo`, recovery выполняется удалением папки UI и повторным стартом `mihomo`.
4. Стартовый маршрут UI переведён с `Proxies` на `Overview` как более безопасную входную точку.
5. На экранах `Хосты 3x-ui` и `Пользователи` убраны формулировки, создававшие ложное впечатление о наличии отдельного service на сервере.
6. Добавлен документ аудита backend-контура и зафиксирован следующий безопасный шаг.

## Что сознательно не делалось
- не добавлялся новый service;
- не трогался подтверждённый recovery/update path;
- не встраивалось server-side хранение без проверки реального backend-контура.
