# v0.6.36 — provider SSL visibility, shared DB snapshot and proxy access control

Что вошло в релиз:

- список SSL-проверок провайдеров возвращён на страницу **«Провайдеры»** в отдельный видимый блок;
- сохранён ручной запуск проверки SSL;
- снимок SSL-провайдеров, время последней проверки и состояние кеша сохраняются в **shared users DB**;
- добавлена таблица LAN-пользователей с хранением **IP / MAC / hostname / source / proxyAccess**;
- добавлен режим доступа к proxy: **allow all** или **allow-list only**;
- ограничение доступа применяется только к **proxy-портам Mihomo**, а не ко всему трафику устройства;
- в agent добавлены команды `blockipports` / `unblockipports` и их сохранение/восстановление после рестарта.

Что важно:

- автоматическую SSL-проверку провайдеров не ломали; её 4-часовой TTL/автообновление сохранены;
- build-fix из `v0.6.35` остаётся в силе: CI падал из-за stale transpiled `src/*.js`.
