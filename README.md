# UIUbuntuXkeen

**UIUbuntuXkeen** — Ubuntu-oriented веб-интерфейс для управления **Mihomo**.

Проект стартует на базе линии **UltraUIXkeen**, но развивается как отдельный продукт под Ubuntu: с нормальной моделью путей, сервисов, фоновых задач и хранения operational state.

## Текущий статус

- Текущая версия линии: **v0.6.9**
- Последний подтверждённо рабочий релиз на сервере: **v0.2.10**
- Текущий шаг: **Tasks provider list visibility and effective SSL source probing**



Релиз **v0.6.9** доводит operational-блок провайдеров в разделе **«Задачи»** до более честного состояния:
- список провайдеров больше не пропадает целиком, если backend checks временно вернули ошибку — предупреждение показывается отдельно, а таблица остаётся доступной;
- direct SSL/TLS probe теперь собирает эффективные URL 3x-ui подписок не только из сохранённой карты, но и из текущего списка proxy providers и backend provider metadata;
- в таблице и мета-тексте закреплена каноничная терминология **«3x-ui подписка / ссылка подписки»**, чтобы Tasks соответствовал реальной архитектуре проекта.
