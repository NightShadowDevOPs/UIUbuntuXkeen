# UIUbuntuXkeen

**UIUbuntuXkeen** — Ubuntu-oriented веб-интерфейс для управления **Mihomo**.

Проект стартует на базе линии **UltraUIXkeen**, но развивается как отдельный продукт под Ubuntu: с нормальной моделью путей, сервисов, фоновых задач и хранения operational state.

## Текущий статус

- Текущая версия линии: **v0.6.8**
- Последний подтверждённо рабочий релиз на сервере: **v0.2.10**
- Текущий шаг: **Tasks real SSL polling by subscription URLs**



Релиз **v0.6.8** впервые подключает реальный SSL/TLS-опрос провайдеров в разделе **«Задачи»**:
- кнопки обновления SSL теперь запускают direct probe по сохранённым URL 3x-ui подписок через compatibility bridge, если он доступен;
- таблица провайдеров показывает реальные даты сертификатов и TLS-ошибки именно по subscription URL;
- stale SSL-результаты очищаются при смене ссылки подписки, чтобы в Tasks не висели старые чужие даты.
