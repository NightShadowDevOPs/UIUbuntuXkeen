# UIUbuntuXkeen

- Текущая версия линии: **v0.6.47**
- Базовый runtime текущего хоста: `mihomo`
- Подтверждённый recovery-механизм: остановить `mihomo`, удалить папку UI, снова запустить `mihomo` — UI скачивается из репозитория автоматически

## Что в v0.6.47

- UI больше не притворяется, что server-side backend уже подключён, если `/api/capabilities` и `/cgi-bin/api.sh?cmd=capabilities` реально отсутствуют или отдают ошибку
- `Хосты 3x-ui` переходят в честный локальный fallback-режим: показывают локально сохранённые URL панелей и отдельное предупреждение о том, что живой SSL backend route не подключён
- `Пользователи` тоже переходят в честный локальный режим с предупреждением, а не рисуют фантомное подключение к shared users DB
- из package scripts убраны старые хвосты от фантомного `ubuntu-service`
