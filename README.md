# UIUbuntuXkeen

- Текущая версия линии: **v0.6.44**
- База: восстановленный рабочий контур через `mihomo` и очистка дистрибутива от роутерных хвостов.
- В этом релизе зафиксирован **реальный runtime contour** проекта: UI обслуживается `mihomo`, корневой `/ui/` уводится на `Overview`, а разделы `Хосты 3x-ui` и `Пользователи` остаются в UI без выдуманного отдельного service.

## Что в v0.6.44
- стартовый маршрут UI переведён на **Overview** вместо `Proxies`;
- выполнен аудит текущего backend-контура проекта и зафиксирован в docs;
- на экранах `Хосты 3x-ui` и `Пользователи` убраны формулировки, которые создавали ложное впечатление о наличии отдельного service на сервере;
- подтверждён recovery/update flow: остановить `mihomo`, удалить папку UI, запустить `mihomo`, дождаться автоматической загрузки UI из репозитория.

## Важное правило
Новые server-side сущности не добавляются в релизы без проверки фактического runtime-механизма проекта. Сначала проверяем, как проект реально обслуживается и обновляется, потом встраиваем функциональность в существующий контур.
