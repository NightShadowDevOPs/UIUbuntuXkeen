# UIUbuntuXkeen

**UIUbuntuXkeen** — Ubuntu-oriented веб-интерфейс для управления **Mihomo**.

Проект стартует на базе линии **UltraUIXkeen**, но развивается как отдельный продукт под Ubuntu: с нормальной моделью путей, сервисов, фоновых задач и хранения operational state.

## Текущий статус

- Текущая версия линии: **v0.6.5**
- Последний подтверждённо рабочий релиз на сервере: **v0.2.10**
- Текущий шаг: **Tasks providers list and form audit**



Релиз **v0.6.5** добивает практический хвост по разделу **«Задачи»**:
- список провайдеров теперь собирается не только из health/checks backend, но и из текущих provider sources UI плюс сохранённых URL/иконок/SSL override-настроек;
- Tasks best-effort подтягивает proxy providers store, поэтому таблица провайдеров не должна пропадать целиком при временно недоступном backend health;
- в `docs/project-spec.md` добавлено прямое описание структуры формы **«Задачи»** и назначения её кнопок/разделов.
