# UIUbuntuXkeen

**UIUbuntuXkeen** — Ubuntu-oriented веб-интерфейс для управления **Mihomo**.

Проект стартует на базе линии **UltraUIXkeen**, но развивается как отдельный продукт под Ubuntu: с нормальной моделью путей, сервисов, фоновых задач и хранения operational state.

## Текущий статус

- Текущая версия линии: **v0.6.4**
- Последний подтверждённо рабочий релиз на сервере: **v0.2.10**
- Текущий шаг: **Tasks alignment: 3x-ui subscription URLs and SSL flow**

Релиз **v0.6.4** возвращает раздел **«Задачи»** к целевой модели со скринов:
- блок провайдеров в Tasks теперь опирается на **3x-ui subscription URL**, а не на путаницу с “панелями”;
- URL подписок можно редактировать прямо в таблице Tasks и в карточке провайдера;
- SSL/TLS статус в Tasks считается по этим ссылкам подписки;
- документация обновлена под фактическую модель: **Providers → subscription URL → Tasks → SSL status**.
