# UIUbuntuXkeen

**UIUbuntuXkeen** — Ubuntu-oriented веб-интерфейс для управления **Mihomo**.

Проект стартует на базе линии **UltraUIXkeen**, но развивается как отдельный продукт под Ubuntu: с нормальной моделью путей, сервисов, фоновых задач и хранения operational state.

## Текущий статус

- Текущая версия линии: **v0.6.6**
- Последний подтверждённо рабочий релиз на сервере: **v0.2.10**
- Текущий шаг: **Tasks providers SSL operational polish**



Релиз **v0.6.6** доводит раздел **«Задачи»** ближе к эталонной operational-форме:
- список провайдеров сохраняет реальный порядок provider sources, а не сортируется в отрыве от UI;
- в блоке провайдеров появились сводные счётчики по ссылкам подписок, SSL-ошибкам и скорым истечениям;
- если backend checks временно недоступен, Tasks отдельно предупреждает об этом, но не прячет локально сохранённые URL 3x-ui подписок.
